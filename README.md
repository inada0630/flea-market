# フリマアプリ

Spring Boot を用いて作成したフリマアプリです。

ユーザー登録・ログイン機能を実装し、
商品の出品、編集、削除、購入ができます。

---

# アプリ概要

ユーザーが商品を出品し、他ユーザーが購入できるフリマアプリを作成しました。

実際のECサイトを意識し、

* 認証
* 権限制御
* 商品管理
* 検索機能
* 購入処理

などを実装しています。

---

# 使用技術

* Java 21
* Spring Boot
* Spring Security
* MyBatis
* MySQL
* Thymeleaf
* Bootstrap
* Gradle

---

# 主な機能

## 認証機能

* ユーザー登録
* ログイン
* ログアウト

## 商品機能

* 商品一覧表示
* 商品詳細表示
* 商品出品
* 商品編集
* 商品削除
* 画像アップロード

## 購入機能

* 商品購入
* 売り切れ管理
* 購入履歴表示
* 販売履歴表示

## 検索機能

* 商品名検索
* 状態検索
* 価格検索

# 工夫した点

* Service 層で権限制御を実装し、自分の商品以外は編集・削除できないようにしました
* 売り切れ商品の編集制限を実装しました
* MyBatis の動的 SQL を利用して複数条件検索を実装しました
* 検索条件を Form クラスで管理し、拡張しやすい構成にしました
* 画像アップロード機能を実装しました
